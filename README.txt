INSTRUCCIONES
1) Abre el archivo index.html en Chrome/Edge.
2) Si no ves gráficas/tabla, revisa tu conexión a internet (este reporte usa librerías por CDN).
3) Usa el buscador y filtros para explorar elevación de COSTO y PMP (Base − 2026).
4) El botón “Descargar CSV (filtrado)” exporta la vista filtrada.

NOTA SOBRE CÁLCULOS
- % Δ Costo = (Costo_Base − Costo_2026) / Costo_2026
- % Δ PMP   = (PMP_Base   − PMP_2026)  / PMP_2026
- Si el valor 2026 está vacío o es 0, no se calcula %.
